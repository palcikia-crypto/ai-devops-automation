# AI DevOps Automation 🤖

Этот репозиторий содержит систему автоматизации DevOps на базе ИИ (Gemini).

## Возможности
* **AI Code Review**: Автоматический аудит Pull Request'ов на безопасность и лучшие практики DevOps.
* **Infrastructure Audit**: Проверка Terraform и Kubernetes манифестов.

## Настройка
1. Добавьте секрет `GEMINI_API_KEY` в настройках этого репозитория (Settings -> Secrets and variables -> Actions).
2. Создайте Pull Request, чтобы запустить AI аудит.
